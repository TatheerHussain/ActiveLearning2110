About


Active Learning is a process whereby students engage in activities, such as reading, writing, discussion, or problem solving that promote analysis, synthesis, and evaluation of class content.


CS 2110 is a computer science lecture course that covers the structure and operation of modern computers that seeks to engage students in understanding basic hardware concepts. Active Learning 2110 is a single page web application that allows the instructors of the class to use questions that guide the lecture and for students to answer timed practice questions that are relevant to the course. This tool seeks to improve both the teaching and learning process during lectures.

Our Values


MISSION: 2B||!2B aims to replace the current tools set in place with a web application that guides lectures with problem sets, engages students, and keeps track of student progress.


VISION: Our Vision is to allow for instructors to engage students in the class with questions so that the instructor can gauge how well students in his or her class are learning. , and at the same time for students to be able to view these practice questions later to review the concepts presented in class.
